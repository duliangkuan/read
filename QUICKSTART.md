# 快速开始

## 安装和运行

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 打开浏览器访问
# http://localhost:3000
```

## 使用说明

### 1. 导入书籍
- 点击"导入书籍"按钮
- 选择txt、word或pdf文件
- 输入书籍名称（可选）
- 上传封面图片（可选）
- 点击"确认导入"

### 2. 阅读书籍
- 在书架页面点击书籍卡片
- 进入阅读页面
- 点击右上角设置按钮可以：
  - 切换背景颜色
  - 切换翻页方式
  - 开启AI辅助阅读

### 3. AI辅助阅读
- 在阅读页面开启AI辅助阅读
- 选择布局模式（左右双栏或上下栏）
- 使用Q-BOOK功能：
  - 点击Q-BOOK按钮
  - 在阅读栏选中文本
  - 在AI对话栏输入问题
  - 点击发送
- 使用Q-TOC功能：
  - 点击Q-TOC按钮
  - 选择章节
  - 在AI对话栏输入问题
  - 点击发送

### 4. 查看AI对话记录
- 在首页切换到"AI书记"页面
- 点击书籍卡片查看对话记录
- 可以导出为Excel或PDF

### 5. 心得帖
- 在阅读页面或AI书记页面
- 点击右下角的心得帖按钮
- 记录阅读心得
- 可以导出为TXT文件

## 功能特点

- 📚 支持多种电子书格式（txt, word, pdf）
- 🤖 基于DeepSeek的AI阅读助手
- 📝 完整的对话记录保存
- 💡 心得帖功能
- 📊 多种导出格式（Excel, PDF, TXT）
- 🎨 多种阅读背景和布局选项

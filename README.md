# AI辅助阅读书籍网站

个人使用的AI辅助精读阅读工具，基于DeepSeek模型。

## 功能特性

- 📚 书籍管理：支持导入txt、word、pdf格式电子书
- 🤖 AI辅助阅读：基于DeepSeek的智能阅读助手
- 📝 对话记录：完整的AI对话历史记录
- 💡 心得帖：记录阅读心得体会
- 📊 导出功能：支持Excel、PDF、TXT导出

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- DeepSeek API

## 开发

```bash
npm install
npm run dev
```

## 部署

部署到Vercel：

```bash
vercel
```
